# Stremio Muhn Pace Addon
A Stremio addon to supplement the existing One Pace addons with the [Muhn Pace fan project](https://steamcommunity.com/sharedfiles/filedetails/?id=3685024934), making it easily available on Android TV's.

## How-to
### Run addon server
Requirements: Node > 22.X, git, pnpm
```sh
git clone https://github.com/Denperidge/stremio-muhn-pace-addon.git
cd stremio-muhn-pace-addon

pnpm install
pnpm start
```

### Generate data files
```sh
pnpm data
# Generates data/meta.json
# & data/subs.json
```

## Explanation
### Compatibility
- It works fully as intended on Android TV & Android
- Aside from the subtitles, works on stremio-linux-shell
- Flatpak/Debian may not work due to libmpv/mp4
- Stremio web may not work due to CORS limitations

### data/ directory
- [links.json](data/links.json) - Links to Muhn Pace episodes & episode names. Generated using [the Pixeldrain Download Links Fetcher](https://github.com/Denperidge/pixeldrain-download-links-fetcher)
- [subs.json.js] - Process the Open Pace subtitle repo for subtitle links & meta info. See [# Generate data files](#generate-data-files)`
- [subs.json](data/subs.json) - Subtitle data for Stremio. Generated by [subs.json.js](data/subs.json.js)
- [meta.json](data/meta.json) - Metadata for Stremio info for individual episodes. Generated by [subs.json.js](data/subs.json.js)
- [onepace.json](data/onepace.json) - Relevant data needed to link to One Pace episodes, as [suggested by the Muhn Pace community post](https://steamcommunity.com/sharedfiles/filedetails/?id=3685024934). Maintained manually
- [$onepace.json](data/$onepace.json) - JSON Schema for [onepace.json](data/onepace.json)
